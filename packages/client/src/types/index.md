# types/

- **models.ts** — `User`, `Server` (includes `allowMemberInvites?: boolean`), `Channel`, `ServerInvitePayload` (`{ code, serverId, serverName }` — shared by `Message`, `DirectMessage`, `SendDmRequest`), `Message`, `CustomToneDefinition` (includes optional animation fields: char, emojiSet, driftDir, matchEmojis), `ServerMember`, `ServerBan`, `Invite`, `Attachment`, `DirectConversation`, `DirectMessage`, `AuditLogEntry` interfaces; `AuditAction` type; `ToneTextStyle` ('normal'|'italic'|'medium'), `CharAnimation` (9 variants: 'bounce'|'tilt'|'lock'|'sway'|'wobble'|'rise'|'sink'|'breathe'|'jitter'), `DriftDir` ('UR'|'U'|'R'|'F') types; `FriendEntry`, `FriendRequest`, `FriendshipStatus` ('none'|'pending_outgoing'|'pending_incoming'|'friends') types
- **api.types.ts** — request types (RegisterRequest, LoginRequest, SendDmRequest (includes optional `serverInvite`), UpdateDmRequest, ToggleDmReactionRequest, UpdateInviteSettingsRequest, etc.), response wrappers (AuthResponse, RefreshResponse, DirectConversationResponse, DirectConversationsResponse, DirectMessageResponse, DirectMessagesResponse, BlockedIdsResponse, FriendsResponse, PendingRequestsResponse, FriendshipStatusResponse, FriendRequestResponse, AuditLogResponse, etc.), query/filter types (MessagesQuery, AuditLogQuery, etc.)
- **socket.types.ts** — `JoinChannelPayload`, `SendMessagePayload`, `MessageDeletedPayload`, `DmSendPayload`, etc.; `ClientToServerEvents` (includes `join_dm`, `leave_dm`, `dm:send`, `dm:typing`, `dm:react`), `ServerToClientEvents` (includes `message_edited`, `message_deleted`, `dm:new_message`, `dm:typing`, `dm:reaction_updated`, `dm:notification`, `friend:request_received`, `friend:request_accepted`) typed event maps
