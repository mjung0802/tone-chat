# types/

- **models.ts** — `User`, `Server`, `Channel`, `Message`, `CustomToneDefinition`, `ServerMember`, `ServerBan`, `Invite`, `Attachment` interfaces
- **api.types.ts** — request types (RegisterRequest, LoginRequest, etc.), response wrappers (AuthResponse, RefreshResponse, etc.), query/filter types (MessagesQuery, etc.)
- **socket.types.ts** — `JoinChannelPayload`, `SendMessagePayload`, etc.; `ClientToServerEvents`, `ServerToClientEvents` typed event maps
