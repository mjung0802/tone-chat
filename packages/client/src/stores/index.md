# stores/

- **instanceStore.ts** — `useInstanceStore` — state: `instances: string[]`, `activeInstance: string | null`, `isHydrated`; persists to localStorage (web) / expo-secure-store (native). All API/socket calls read `activeInstance` at runtime. URLs normalized (trailing slash stripped); `addInstance()` deduplicates. Actions: `addInstance`, `setActiveInstance`, `clearActiveInstance` (nulls active without removing from saved list — used by switch-instance flow), `removeInstance`, `hydrate`. Must be hydrated before `authStore`.
- **authStore.ts** — `useAuthStore` — state: accessToken, refreshToken, userId, isAuthenticated, emailVerified, isHydrated; tokens scoped per instance URL (keys: `accessToken:${instance}`, `refreshToken:${instance}`); on native: persists both tokens to expo-secure-store; on web: persists only accessToken to localStorage (refreshToken stored in httpOnly cookie by BFF, never in JS storage). `loadInstanceTokens(url)` switches the active token set.
- **socketStore.ts** — `useSocketStore` — Socket.IO lifecycle: connect, disconnect, updateToken; BFF URL read from `instanceStore.activeInstance` at connect time; subscribes to instanceStore and reconnects if `activeInstance` changes
- **notificationStore.ts** — `useNotificationStore` — mention notification state (currentNotification, currentChannelId), quiet/system preferences; DM unread tracking (dmUnreadEntries: `Record<conversationId, { count, otherUserId }>`), incrementDmUnread(), clearConversationUnread(), clearAllDmUnreads(); selector `selectTotalDmUnread(state)` returns total across all conversations; `hydrateNotificationPreference()`
- **uiStore.ts** — `useUiStore` — theme preference (light/dark/system), tone display (full/reduced), color theme selection, sidebar state, friends view state (`isFriendsViewOpen`, `friendsTab: 'friends'|'pending'`); `hydrateTheme()`, `hydrateToneDisplay()`, `hydrateColorTheme()`, `hydrateUiStore()`, `openFriendsView()`, `closeFriendsView()`, `setFriendsTab(tab)`
- **authStore.test.ts** / **notificationStore.test.ts** / **uiStore.test.ts** — unit tests
